2.9. Use options

One last preliminary. Inform has a number of optional settings which affect the result of translating the source. The sentence:

Use American dialect.

makes the resulting work of IF use American spellings (except where the designer spells otherwise) and the American convention for printing numbers (thus, "one hundred seventeen" not "one hundred and seventeen"). Similarly:

Use the serial comma.

uses a comma in lists: thus "Julian, Dick, George, and Anne" rather than "Julian, Dick, George and Anne". Lastly, for the benefit of Inform 6 users or for projects which overflow Inform 6's normal limits, we can specify settings like:

Use MAX_ZCODE_SIZE of 60000.

(or any of I6's other memory settings): the effect is to raise the setting if need be, but never to lower it.

We can combine any number of options in a single "Use" sentence, so for example:

Use American dialect and the serial comma.

brings about both of these changes.

This completes the groundwork, and we are ready to begin on simulating a world to explore.

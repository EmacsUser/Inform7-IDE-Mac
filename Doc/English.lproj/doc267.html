18.4. Inverted assertions

Because of a curious irregularity in English, it is not quite true that every assertion could also be used as a condition.

The irregularity is that we sometimes reverse a sentence so that what Inform would call the Verb is split up. For instance,

In the Garden is a sunflower.

This does not follow the pattern S = DP + VP, because "in the garden" is not a DP: indeed, it is not a noun at all. To make sense of this sentence, Inform reverses it so that it does fit the pattern:

A sunflower is in the Garden.

This allows a whole load of new sentences which our grammar does not at present fit, and we can widen our grammar by adding the following rule:

6. S = RP is/are DP

But we need to remember that only assertions can do this, so "if in the garden is a sunflower..." is not allowed as a condition.

(This quirk of English is called subject-verb inversion: see the Oxford English Grammar at 3.22F.)

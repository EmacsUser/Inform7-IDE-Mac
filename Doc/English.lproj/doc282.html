18.10. Accompanying a release with a website

The final touch is to use Inform to manufacture a suitable web page which will act as a cover page for the work. We request this by writing:

Release along with cover art, a website, a file of "Collegio magazine" called "Collegio.pdf" and a file of "The mating call of the green wyvern" called "Mating Wyverns.mp3".

where the list of ingredients now includes "a website". In fact, Inform makes only a single web page, called "index.html", which it places in the materials folder (as set up in the previous section): this then contains suitable links to all the other material. For instance:

Once this is done, take the Blorb file produced by Inform, rename it "story.zblorb" and move it into the Materials folder. This folder now contains everything needed:

Works in Progress
    Magician Materials
        Collegio.pdf
        Cover.jpg
        index.html
        Mating Wyverns.mp3
        Small Cover.jpg
        story.zblorb
    Magician.inform

Finally, then, the whole folder here called "Magician Materials" can be put online as part of a website, or can be zipped into a single ".zip" archive and emailed out, or uploaded to the IF Archive. (It can be renamed as anything we like: no need for it to still be called "Magician Materials".)

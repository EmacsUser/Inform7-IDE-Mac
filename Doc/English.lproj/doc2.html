1.2. Getting started

These notes are arranged so that the reader can, in principle, write whole works of IF as early as the end of chapter 2; each subsequent chapter then extends the range of techniques available to make livelier and more intriguing situations.

At the start we have only a blank page in which to write the first lines of a new interactive fiction - the Source - and this panel, the Documentation. Clicking the Go button translates the text in the Source panel into a computer program which enacts the interactive fiction, and automatically sets it going (in a new panel which appears, called "Game"). If the Source is empty of text, Inform will be unable to create anything: it needs at least one name of a location where the drama can unfold. For reasons of tradition, such locations are normally called "rooms", though people have used them to represent anything from grassy fields to states of mind and other metaphorical places.

"Midsummer Day"

The Gazebo is a room.

Throughout these notes, when an example of Inform source begins with a title in quotation marks (here "Midsummer Day"), it is complete and needs no other source text to work: whereas examples which do not begin with titles are only extracts.

Clicking Go with this source will result in the right-hand panel switching to the Game, where we can explore this newly created model world: an interactive fiction called "Midsummer Day". It will not be very exciting, since Inform has only five words to go on, but we can add more detail to the source at any point and then click Go again to try out the changes. (Note that there is no need to "quit" these explorations in the Game panel. When Go is clicked, any game already in progress is discarded in favour of the new version.)

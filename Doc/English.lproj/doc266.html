18.3. Relative phrases RP

"Which" and "who" can be concisely defined by the following extensions to our sentence grammar:

3a. DP = DP which VP
3b. DP = DP who VP

Thus "an open container which is in the Ballroom" can be broken down as:

DP (an open container) which VP (is in the Ballroom)

To understand compounds like "something in a container", we have to invent a new grammatical structure: let's call this a Relative Phrase, or RP.

4. DP = DP + RP

Thus "an open container in the Ballroom" is DP (an open container) + RP (in the Ballroom).

Relative Phrases have two different forms:

5a. RP = preposition + DP
5b. RP = participle + DP

Examples of these would be:

RP (on the table) = Preposition (on) + DP (the table)
RP (worn by Mr Darcy) = Participle (worn by) + DP (Mr Darcy)

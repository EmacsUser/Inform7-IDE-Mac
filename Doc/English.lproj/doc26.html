2.16. Possessions and clothing

We have seen how to place objects in rooms, and in containers or on supporters, but not how to make them begin in the player's possession, which we do as follows:

A brass lantern and an old brass key are initially carried.

"Initially carried" is a special either/or property which only affects the beginning of the game.

Clothes are simply things which have the either/or property of being "wearable": if something has this property then the player will be allowed to wear it, provided it can first be picked up. It will then have the either/or property of being "worn". To start the player off wearing something, we need only say that it is worn:

The cardinal's hat is worn.

This is because Inform assumes that anything said to be "worn" at the start of play is also likely to be "wearable" and "initially carried". We can override these assumptions by writing something definite to the contrary:

The cardinal's hat is worn. The hat is not wearable.

The result would be a hat which could be removed but not put on again, so it is unlikely that we would want this very often.

16.10. Printing the plural name of something

1. When it happens. Only when a group of identical items is present in the same place, and are being described jointly with text like "You can see five gold rings here." The action happens after "five" and before "here."

2. The default behaviour. The plural name - in this case "gold rings" - is printed out.

3. Examples. See below.

Example (**). Letters which are described differently as a group, depending on whether the player has read none, some, or all of them, and on whether they are alike or unlike.

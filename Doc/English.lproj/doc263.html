Chapter 18: Appendix: The Sentence Grammar

18.1. The sentence grammar S

The Inform documentation has tried not to be too formal about grammar, to avoid spelling out in technical language what the reader already knows perfectly well about everyday English. All the same, some readers will want to know exactly what is Inform's idea of a sentence, or indeed a noun, and the following short account gives a (very nearly) exact description. It is intended only to explain why some things are accepted and others not. No new features of the language are described, and this is very much an appendix, not part of the manual proper.

Let us start with two major grammatical structures. One is the Description Phrase, or DP for short, examples of which include "the red basket", "somewhere lighted" and "an empty open container". This we will take as read, since the account given at the start of Chapter 5 is quite explicit. The other is the Sentence, which we will write S, an example of which is:

The horseman wears a riding helmet.

We have been much vaguer about the definition of "sentence". Clearly sentences include DPs, but they also include other ingredients. How does this work?

The general pattern used in Inform is very simple:

1. S = DP + VP
2. VP = Verb + DP

where VP is a "Verb Phrase". Using these two rules, we can break down the above example sentence like so:

S (The horseman wears a riding helmet)
    = DP (The horseman) + VP (wears a riding helmet)

and then:

VP (wears a riding helmet)
    = Verb (wears) + DP (a riding helmet)

21.4. Emily Short - My Nose

Reported by: Emily Short / Date: 1 October 2005

Nature: The source looks legal and produces a playable game, but the game does not do what it should

Build: 2Z31

Platform: Mac OS X

Code example:

"Nasal Twinge 2"

Sainsbury's is a room. The player is carrying a box of chocolates and an Invisible Nose.

Rule for deciding the concealed possessions of the player: if the particular possession is the Invisible Nose, yes; no.

What goes wrong:

>i

You are carrying:

a box of chocolates

Invisible Nose

>drop nose

You can't see any such thing.

>x nose

You can't see any such thing.

Comments:

This seems like a surreal way for things to work. I realize that you probably didn't intend for the player's own possessions to be able to be concealed, but hey, it's my job to think of these things, etc.

Miraculous you still speak to me, when I think about it.

6.6. Daemons!

A daemon is a rule which is applied every turn, but only when it is active: the rest of the time it sleeps. Traditionally, rules like this in interactive fiction have been imagined as being somehow independently acting, hence the term "daemon". There's not much to say about daemons:

The lightning daemon: say "Lightning strikes, suddenly."

In the Keep is a lever. The lever is fixed in place. Instead of pulling the lever: say "Zzzing..."; start the lightning daemon. Instead of pushing the lever: say "Zzzap..."; stop the lightning daemon.

The name of a daemon must always have "daemon" as its final word. When play begins, daemons are normally asleep, so if we need one of them to be active right from the start then we must add a further rule:

When play begins, start the lightning daemon.

1.8. The mouseover buttons in the Skein

Moving the mouse over a command in the Skein panel causes a cluster of little icon buttons to appear:

The skein is a new feature to Inform and is unfinished, so not all of these buttons do anything as yet. Clockwise from top left:

Annotate - Not yet implemented, but will allow notes to be added

Transcript - Not yet implemented, but will allow game transcripts to be compared with earlier versions

Delete - Delete this command, and all threads hanging from it: not displayed for the start knot, for obvious reasons

Add - Add a command here

Unlocked (amber, padlock open) - This command is locked, and cannot be deleted until it is unlocked; or

Locked (green, padlock closed) - This command is unlocked, and can freely be deleted
